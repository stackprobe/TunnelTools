----
■使用例(1)

以下のコマンドを実行します。

	> SmplProxy

ブラウザのプロキシサーバーを localhost:8080 に変更して下さい。
ブラウザを開くと、HTTPProxy.exe を経由してインターネットにアクセスしているはずです。

----
■使用例(2) -- ダウンロード画像を差し替えるHTTPプロキシ

以下のコマンドを実行します。

	> SmplProxy_FaceFilter

ブラウザのプロキシサーバーを localhost:8080 に変更して下さい。
ブラウザを開くと、HTTPでダウンロードした画像が sample\0005\Face.png に差し替わっているはずです。

----
■使用例(3) -- ダウンロードしたコンテンツをファイルに保存するHTTPプロキシ

以下のコマンドを実行します。

	> SmplProxy_SmplSaver

ブラウザのプロキシサーバーを localhost:8080 に変更して下さい。
ブラウザを開くと、HTTPでダウンロードしたコンテンツが "ダウンロード・コンテンツ" に保存されるはずです。

	★カレントディレクトリに "ダウンロード・コンテンツ" というディレクトリが無ければ作成します。

----
■ポート番号

	ポート番号は 8080 を使用します。
	ポート番号を変更するには SmplProxy.bat の HTTPProxy コマンドの第１引数を変更して下さい。

----
■制限というか問題点

	HTTPで巨大なファイルをダウンロードする場合、クライアント側のダウンロード開始が大幅に遅れます。
